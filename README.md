# 🎮 Caro Plus

Caro Plus is an online **Gomoku (Caro)** game built using Java Spring Boot.
The system allows players to create matches, play in real-time, and chat with each other.

This project is developed as a course assignment.

---

# 🚀 Features

* 👤 User management (register / login)
* 🎮 Create and join game rooms
* ♟️ Real-time Caro gameplay
* 💬 Chat while playing
* 📜 Match history tracking
* 🧑‍💻 Multiplayer support

---

# 🛠️ Tech Stack

Backend:

- Java
- Spring Boot
- Spring WebSocket (for real-time gameplay)
- REST API

Database:

- MySQL / PostgreSQL

Other Tools:

- Git & GitHub
- Maven / Gradle

# ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/trnghuy/caro_plus.git
```

2. Go to project folder

```
cd caro-plus
```

3. Run the project

```
mvn spring-boot:run
```
